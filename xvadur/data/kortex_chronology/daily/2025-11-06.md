### **Ako je Štruktúrovaný Môj Backup: Hypotéza**

Hoci nevidím presnú štruktúru súboru z exportu, môžem s vysokou mierou pravdepodobnosti predpokladať, ako vyzerá. Bude to pravdepodobne súbor vo formáte **JSON** (alebo variant ako JSONL), čo je štandard pre export štruktúrovaných dát.

Bude to vyzerať asi takto – buď ako **jeden obrovský JSON objekt**, alebo (čo je pravdepodobnejšie a lepšie pre spracovanie veľkých súborov) ako **súbor JSONL (JSON Lines)**, kde každý riadok je samostatný JSON objekt.

**Príklad štruktúry (zjednodušený):**

```json
[
  {
    "conversation_id": "conv_123",
    "timestamp": "2024-11-03T12:00:00Z",
    "messages": [
      {
        "role": "user",
        "content": "toto je moj prvy prompt..."
      },
      {
        "role": "assistant",
        "content": "odpoved od kAI..."
      }
    ]
  },
  {
    "conversation_id": "conv_124",
    "timestamp": "2024-11-03T13:00:00Z",
    "messages": [
      {
        "role": "user",
        "content": "toto je moj druhy prompt..."
      },
      {
        "role": "assistant",
        "content": "dalsia odpoved..."
      }
    ]
  }
]
```

**Tvoj cieľ:** Z tohto celého súboru potrebuješ extrahovať iba hodnotu z poľa `"content"`, kde hodnota poľa `"role"` je `"user"`.

#### **Verzia A: Ak je backup jeden veľký JSON zoznam (array)**

Táto metóda je náročnejšia na pamäť, ale existujú knižnice, ktoré to vedia spraviť inteligentne. Použijeme knižnicu `ijson`,  ktorá je navrhnutá presne na toto.

Najprv si ju nainštaluj. V termináli napíš:
`pip install ijson`

A tu je kód pre `extract.py`:

```python
import ijson

input_file = 'backup.json'
output_file = 'moje_prompty.txt'

print(f"Spracovávam súbor: {input_file}...")

# Otvoríme výstupný súbor na zápis
with open(output_file, 'w', encoding='utf-8') as outfile:
    # Použijeme ijson na postupné prechádzanie JSON súborom
    # Predpokladáme, že štruktúra je [{...}, {...}]
    # 'item' tu odkazuje na každý objekt v hlavnom zozname
    conversations = ijson.items(open(input_file, 'rb'), 'item')
    
    for conv in conversations:
        # Skontrolujeme, či konverzácia obsahuje 'messages'
        if 'messages' in conv and isinstance(conv['messages'], list):
            for message in conv['messages']:
                # Skontrolujeme, či správa má správnu štruktúru
                if 'role' in message and 'content' in message:
                    # Ak je to prompt od užívateľa, zapíšeme obsah
                    if message['role'] == 'user':
                        # Pridáme oddeľovač, aby sme vedeli, kde končí jeden prompt a začína ďalší
                        outfile.write(message['content'] + "\n\n---\n\n")

print(f"Hotovo! Všetky tvoje prompty boli uložené do súboru: {output_file}")
```

#### **Verzia B: Ak je backup súbor vo formáte JSONL (jeden JSON na riadok)**

Toto je  oveľa jednoduchšie a pamäťovo efektívnejšie. Na toto nepotrebuješ ani externú knižnicu.

```python
import json

input_file = 'backup.jsonl' # Uisti sa, že máš správnu príponu, môže byť aj .json
output_file = 'moje_prompty.txt'

print(f"Spracovávam súbor: {input_file}...")

# Otvoríme výstupný súbor na zápis
with open(output_file, 'w', encoding='utf-8') as outfile:
    # Otvoríme vstupný súbor na čítanie
    with open(input_file, 'r', encoding='utf-8') as infile:
        # Čítame súbor riadok po riadku
        for line in infile:
            try:
                # Každý riadok parsujeme ako samostatný JSON
                data = json.loads(line)
                
                # Skontrolujeme, či konverzácia obsahuje 'messages'
                if 'messages' in data and isinstance(data['messages'], list):
                    for message in data['messages']:
                        # Skontrolujeme, či správa má správnu štruktúru
                        if 'role' in message and 'content' in message:
                            # Ak je to prompt od užívateľa, zapíšeme obsah
                            if message['role'] == 'user':
                                outfile.write(message['content'] + "\n\n---\n\n")
            except json.JSONDecodeError:
                # Ignorujeme riadky, ktoré nie sú validným JSON
                print(f"Chyba pri spracovaní riadku, preskakujem: {line.strip()}")

print(f"Hotovo! Všetky tvoje prompty boli uložené do súboru: {output_file}")
```
